lista libres : Boolean
es una lista que representa las fichas que no fueron colocadas

lista fichas : Ficha
es una lista con el total de las fichas. La intencion es que se corresponda uno a uno con la lista de libres.
Cada ficha tiene un id empezando desde 0 que se le asigna segun el orden en que fue creada. 

libres :{t,t,t,t}
fichas :{f1,f2,f3,f4}
